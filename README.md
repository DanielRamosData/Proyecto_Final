# 🛡️ Análisis de Renovación de Pólizas - Insurance Company

Este repositorio contiene un **Producto Analítico** interactivo desarrollado en Python para identificar los factores clave que influyen en la renovación de seguros. El proyecto utiliza un enfoque de **Análisis Exploratorio de Datos (EDA)** exhaustivo para extraer *insights* de negocio, cumpliendo con los estándares de calidad de la especialización.

## 📊 Características del Proyecto
- **Arquitectura POO:** Lógica de procesamiento y análisis encapsulada en la clase `DataAnalyzer` para asegurar un código escalable y organizado.
- **Interfaz Interactiva:** Desarrollada con **Streamlit**, permitiendo una navegación fluida entre módulos de carga, análisis y conclusiones.
- **Análisis Visual Avanzado:** Implementación de gráficos univariados y bivariados utilizando **Seaborn** y **Matplotlib** para detectar patrones de morosidad e ingresos.
- **Enfoque en Toma de Decisiones:** 5 conclusiones técnicas orientadas a mejorar la retención de clientes sin el uso de modelos predictivos.

## 🛠️ Stack Tecnológico
- **Lenguaje:** Python 3.x
- **Librerías principales:** Pandas, NumPy, Matplotlib, Seaborn.
- **Framework Web:** Streamlit.

## 📁 Estructura del Repositorio
- `app.py`: Código fuente principal de la aplicación.
- `requirements.txt`: Lista de dependencias necesarias para el despliegue.
- `InsuranceCompany.csv`: Dataset original de estudio.
